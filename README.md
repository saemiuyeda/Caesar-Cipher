Caesar Cipher

Este projeto implementa o algoritmo de criptografia Caesar Cipher permitindo encriptar e decriptar mensagens a partir de um deslocamento definido pelo usuário.

Algotigmo de criptografia Caesar Cipher: 
É uma técnica de substituição em que cada letra da mensagem é deslocada um número fixo de posições no alfabeto.

Funcionalidades:
- O usuário escolhe entre encriptar ou decriptar o texto
- Permite definir o valor de deslocamento
